# FlowTask-style SaaS Frontend - Master Outline
## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Core Sections (Landing Page)
1. **Hero** – animated gradient, CTA
2. **Features** – grid with icons, glassmorphism cards
3. **How It Works** – steps with animations
4. **Pricing** – 3 tiers, recommended badge
5. **Testimonials** – carousel
6. **FAQ** – accordion
7. **Footer**

## Dashboard (Authenticated)
- Responsive sidebar (collapses on mobile)
- Header with user menu
- Page shells for each module
- Dark/light mode toggle (persists)

## Design Language
- Glassmorphism elements
- Mobile-first, touch-friendly
- Subtle micro-interactions (Framer Motion)
- Dark mode default
